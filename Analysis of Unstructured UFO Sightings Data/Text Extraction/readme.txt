The text_extract.jar is the one to process text extraction. Call this like:
 java -jar text_extract.jar <directory-of-text-files-need-extract> <file-to-append>

The words.txt is the dictionary used in text_extract.jar to filter proper words from the messy text file. This file should be put in the same directory with text_extract.jar.

Main.java is java source code of text_extract.jar.
